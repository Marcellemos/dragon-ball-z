# Projeto Dragon Ball Z

Este é um projeto web estático que parece exibir conteúdo de Dragon Ball Z usando HTML, CSS e JavaScript.

## Estrutura do Projeto

- `index.html` — entrada principal do site.
- `assets/` — contém recursos de mídia, como imagens e vídeos.
- `css/` — contém folhas de estilo:
  - `styles.css`
  - `jornada.css`
- `js/` — contém módulos e scripts JavaScript:
  - `hero.js`
  - `jornada.js`
  - `personagens-blueprint.js`
  - `script.js`

## Como Usar

1. Abra `index.html` em um navegador.
2. A página deve carregar os estilos e o JavaScript automaticamente.
3. Se quiser editar o projeto, atualize os arquivos HTML, CSS ou JS e atualize o navegador.

## Observações

- Este é um projeto front-end e não requer servidor para rodar.
- Para desenvolvimento, usar um servidor estático local pode ajudar a evitar restrições do navegador para alguns recursos JavaScript.
